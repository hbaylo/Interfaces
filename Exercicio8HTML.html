Exemplo 8 HTML - Uso das tags <ol>, <ul> e <li>

As tags <ol>,<ul> representão uma lista em html, e são utilizadas para as mais diversas finalidades. A tag <ol> representa uma lista ordenada, já a tag <ul> representa uma lista não ordenada.

Em ambas as listas para representar os elementos da lista usamos a tag <li>

Além disso, estas listas podem conter outras listas de forma alinhada

Etapas para Desenvolver um site(Lista Ordenada)

  1. Planejamento
    1. Definir o objetivo do site
    2. Identificar o publico alvo
    3. Criar um mapa do site (site map)
  2. Desing
    1. Criar wireframes e protótipos
    2. Definir a paleta de cores e tipografia
    3. Desenvolver o layout no figma ou Adode XD
  3. Desenvolvimento
    1. Configurar o amvbiente de desenvolvimento
    2. Codificar o front-end(thm, css e javascript)
    3. Implementar o back-end, com linguagens orientadas a objeto(C#, Java, PHP, Python)
  4. Testes
    1. Testar a responsividade em diferentes dispositivos
    2. Realizar teste de usabilidade
    3. Corrigir bugs identificados
  5. Publicação
    1. Escolher um provedor de hospedagem
    2. Configurar o domínio e o SSL(a segurança)
    3. Fazer o deplay do site
