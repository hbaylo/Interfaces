Estrutura de um documento HTM

A estrutura de um documento HTML é definida pela declaração !DOCTYPE html, pelas tags html, head e body.

Essa estrutura é interpretada pelo navigador que analisa cada tag e rederiza para o usuário.

A tag head é utilizada para configurações do navegador e consequentemente do documento HTML, as informações contidas dentro dessa tag são invisíveis para o usuário, com exeção do titulo que ficara visivel na parte superior da página

Já a tag body é utilizada para exibir toda a parte visível para o usuário e nela teremos outras tags que também serão utilizadas pelo navegador.

Tag HTML, essa tag é a raiz que contém todas as outras tags do documento html, delimitando a totalidade do documento.

Por fim a instrução DOCTYPE html, que tem a finalidade de indicar ao navegador qual a versão do html que será utilizado. Ela deve vir na primeira linha do documento.
