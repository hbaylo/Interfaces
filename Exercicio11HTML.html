Exemplo 11 HTML - Formulários

Os formulários em HTML são um dos principais recursos para a coleta de dados, eles devem ser simples e altamente funcionais, atendendo sempre as necessidades dos usuarios

Para trabalhar com formulários em HTML, possuimos uma série de componenetes que vão facilitar o processo de interação com o usuario. Todos os formulários começam necessariamente com a tag <form>, essa tag indica para o navegador que a página html possui um formulário que pode enviar dados.

Além da tag <form> temos também que usar os atributos action( que define o local - url - em que os dados recolhidos/coletados devem ser enviados ), e o method que define qual o método http para enviar os dados GET ou POST
